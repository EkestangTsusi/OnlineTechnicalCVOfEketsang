Puzzle Game

Puzzle Game • March 2020 - June 2020

The Puzzle Sort Game is a very simple, but challenging 3 x 3 game designed to challenge a person’s problem solving skills including logic and pattern. The aim of the game is for the user to sort the tiles in ascending order while trying to do this with the least number of moves as possible to score maximum points. Once the tiles have been arranged in the correct ascending order the number of moves to complete the tiles is displayed and option to start a new game is presented. Technologies Used Python and Visual Studio Code.
